# 管理后台功能测试指南

## 访问地址
`http://localhost:3002/admin`

## 测试清单

### 1. 页面加载测试 ✓
- [ ] 页面正常加载，无错误
- [ ] 显示提示词列表
- [ ] 显示统计信息（如果有数据）
- [ ] 搜索框可见且可用

### 2. 创建功能测试 (Create)

#### 步骤：
1. 点击右上角的"新建提示词"按钮
2. 观察表单是否自动滚动到视图中
3. 检查表单是否显示蓝色左边框（表示新建模式）
4. 检查表单标题是否为"新建提示词"

#### 测试用例 A：正常创建
- [ ] 填写以下字段：
  - 效果目标：测试提示词
  - 来源：测试来源
  - 标签：测试, AI, 提示词
  - 详细描述：这是一个测试提示词
  - 提示词内容：你是一个测试助手
  - 图片 URL：（可选）
- [ ] 确认所有输入框都是白色背景（可编辑状态）
- [ ] 点击"创建"按钮
- [ ] 确认显示成功提示
- [ ] 确认表单自动关闭
- [ ] 确认新数据出现在列表顶部

#### 测试用例 B：验证测试
- [ ] 打开新建表单
- [ ] 只填写部分必填字段
- [ ] 确认"创建"按钮被禁用
- [ ] 填写所有必填字段
- [ ] 确认"创建"按钮变为可用

#### 测试用例 C：取消操作
- [ ] 打开新建表单
- [ ] 填写一些数据
- [ ] 点击右上角的 X 或"取消"按钮
- [ ] 确认表单关闭
- [ ] 确认数据未被保存

### 3. 读取功能测试 (Read)

#### 基本显示
- [ ] 表格显示所有提示词
- [ ] 每行显示：效果目标、详细描述、标签、更新时间、操作按钮
- [ ] 标签显示为彩色徽章
- [ ] 更新时间格式正确

#### 搜索功能
- [ ] 在搜索框中输入关键词
- [ ] 确认列表实时过滤
- [ ] 测试搜索：
  - [ ] 按效果目标搜索
  - [ ] 按描述搜索
  - [ ] 按标签搜索
- [ ] 清空搜索框，确认显示所有数据

### 4. 更新功能测试 (Update)

#### 步骤：
1. 在列表中找到一个提示词
2. 点击该行的"编辑"按钮（蓝色铅笔图标）
3. 观察表单是否自动滚动到视图中

#### 测试用例 A：正常编辑
- [ ] 表单显示绿色左边框（编辑模式）
- [ ] 表单标题为"编辑提示词"
- [ ] 标题下方显示"正在编辑: {原效果目标}"
- [ ] 所有字段都正确填充了原数据
- [ ] 所有输入框都是白色背景（可编辑状态）
- [ ] 修改一些字段
- [ ] 点击"保存更改"按钮
- [ ] 确认显示成功提示
- [ ] 确认表单关闭
- [ ] 确认列表中的数据已更新

#### 测试用例 B：编辑后取消
- [ ] 点击编辑按钮
- [ ] 修改一些数据
- [ ] 点击"取消"按钮
- [ ] 确认表单关闭
- [ ] 确认原数据未被修改

#### 测试用例 C：多次编辑不同项
- [ ] 编辑项目 A
- [ ] 点击取消
- [ ] 编辑项目 B
- [ ] 确认表单显示项目 B 的数据（不是项目 A）
- [ ] 点击取消
- [ ] 再次编辑项目 A
- [ ] 确认表单显示项目 A 的数据

### 5. 删除功能测试 (Delete)

#### 测试用例 A：正常删除
- [ ] 找到一个要删除的提示词
- [ ] 点击"删除"按钮（红色垃圾桶图标）
- [ ] 确认只显示一次确认对话框（不是两次）
- [ ] 点击"确定"
- [ ] 确认显示成功提示
- [ ] 确认该项从列表中消失

#### 测试用例 B：取消删除
- [ ] 点击"删除"按钮
- [ ] 在确认对话框中点击"取消"
- [ ] 确认数据未被删除
