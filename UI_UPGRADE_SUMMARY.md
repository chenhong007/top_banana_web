## 设计特点

### 配色方案
- **主色调 (Primary)**: 金色 - 代表高端、专业
- **辅助色 (Secondary)**: 蓝色 - 代表科技、创新
- **强调色 (Accent)**: 紫色 - 代表创意、魔法
- **背景**: 深黑色 - 提供现代感和对比度

### 视觉效果
1. **玻璃态设计**: 半透明背景 + 背景模糊 + 精细边框
2. **发光效果**: 悬停时出现的柔和光晕
3. **渐变动画**: 流动的渐变色彩
4. **浮动效果**: 轻微的上下浮动增加活力
5. **淡入动画**: 页面元素依次淡入，提升体验

### 交互体验
- 所有按钮和卡片都有流畅的悬停效果
- 选中状态清晰明显
- 动画持续时间适中（300-500ms）
- 使用 ease-out 缓动函数
